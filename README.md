# 项目启动

## 前端

1. 环境

| 工具   | 版本号     |
|------|---------|
| npm  | 8.6.0   | 
| node | &gt;=18 | 

2. 启动

```shell
cd .\web
```

```shell
npm ci
```

```shell
npm run dev
```

## 后端

| 工具  | 版本号     |
|-----|---------|
| jdk | &gt;=17 |

> 将 aspose-words-21.4.0-jdk17.jar 放到[此处](src/main/resources/lib)
